# UTAU-Uial-Frontend-TTS
- Can load UTAU songs.
- Requires 3rd party gits I have no control on.
- Uses some components from my multilingual kokoro frontend.
- Provided with scripts to get voices and gits and also to download requirements.
- Extra script to download requirements and apply Spacy specifics.
- It's not ready for production and I try to get better pronounciation on English text.
- Tool is meant to be for text to speach clipboard reading so it needs optimisation.
- It is actually quite good at importing utau project files and I kinda recognoze the sounds more than in English.
